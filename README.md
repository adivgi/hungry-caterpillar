# The Very Hungry Caterpillar

A Snake game and presentation inspired by Eric Carle's *The Very Hungry Caterpillar*.

## Play

🎮 [Play the game](https://adivgi.github.io/hungry-caterpillar/)

## Contents

- **Snake Game** (`index.html`) — guide the caterpillar to eat fruit and junk food, grow bigger, and become a butterfly 🦋
- **Presentation** (`hungry-caterpillar-presentation.html`) — an interactive slideshow based on the story

## Features

- 🍎 Eat healthy fruit and junk food from the book
- 📈 6 speed levels — gets harder as you score
- 🏆 Leaderboard with player names and fastest win times
- 🧱 Toggle between walls on/off
- 📱 Fully mobile-friendly with on-screen controls

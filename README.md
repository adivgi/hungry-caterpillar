A Snake game and presentation inspired by Eric Carle's *The Very Hungry Caterpillar*.

## Contents

- **Snake Game** — a Snake game where you play as the hungry caterpillar, eating your way through Eric Carle's colorful world
- **Presentation** — an interactive slideshow based on the story
